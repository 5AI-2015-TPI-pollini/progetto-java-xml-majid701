Abdul Weather Finder

Abdul Weather finder is a free software that retrives weather informations of a place typed by the user. It's easy and free to use! Download the executable or fork the project on GitHub

How to install?

Just download the jar file and run it. Make sure java is installed on your computer

How to use?

Type in a location name and press search. And voil√† the weather information will be displayed on the screen instantly.

Need a proxy?

No problem click on the menu edit>proxy and configure a proxy server. Dont't forget to test the connection ;)
